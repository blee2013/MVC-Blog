# Back-End E-Commerce 

## Description
* Tech Blog where users are able to post and comment on blogs. 

## Tools Used to Build
* dotenv
* Express
* Express-Session
* Heroku
* Insomnia
* JavaScript
* MySQL
* MySQL2.js
* Node.js
* Node.bcrypt.js
* Express Handlebars


## Instructions for Installation
* npm install
* use env to login to SQL
* mysql source db/schema.sql
* npm start
* npm run seed
